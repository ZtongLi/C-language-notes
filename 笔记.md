### 8.1 输出函数printf

- 语法： printf(格式声明，输出表列)

- 格式字符：

% c输出一个字符

%f 用小数的形式输出实数

```
%-m.nf
```

用来控制浮点数的精度

-是左对齐的标志：左对齐和右对齐的区别在于，当内容比字段宽度短的时候，空格放在哪边

字段宽度m： 表示整个输出占多少字符的位置

精度n： 表示小数点后字符串的长度

```
// 右对齐（默认）：空格在左边，数字靠右
printf("[%10.2f]\n", num);
// 输出: [      3.14]
```
```
// 左对齐（加-）：空格在右边，数字靠左  
printf("[%-10.2f]\n", num);
// 输出: [3.14      ]
```
%e e格式符以指数形式输出字符

指数部分占5列，小数点前必须有且仅有1位数字
```
printf(”%e”,123.456);
```

输出1.234560 e+002

%m.ne
```

```

|    特性    |     `putchar`     |           `printf`          |
| :------: | :---------------: | :-------------------------: |

# 结构设计

## 1. 顺序结构设计

- 例1： 给出三角形的三边长，裘三角形的面积

思路： s = 1/2（a+b+c）

```
#include <stdio.h>
#include <math.h>
int main ( )
{ double a,b,c,s,area; 
 a=3.67; 
 b=5.43; 
 c=6.21; 
 s=(a+b+c)/2; 
 area=sqrt(s*(s-a)*(s-b)*(s-c)); 
printf("a=%f\tb=%f\t%f\n",a,b,c); 
 printf("area=%f\n",area); 
 return 0;
}
```

其中
`\t` 
为转义字符，使输出的位置跳到下一个tab位置

- 形式2:  双分支else结构

```

```
这里的表达式，可以是**关系表达式**，**数值表达式**，也可以是**逻辑表达式**

这里的语句，可以是**简单的语句**，**复合语句**，也可以是**另一个if语句**

- 例：冒泡排序
```
#include <stdio.h>
void main()
{ float a,b,c,t;
 printf("please enter a,b,c:");
 scanf("%f,%f,%f",&a,&b,&c);
 if(a<b)
 { t=a; a=b; b=t; } 
 if(a<c)
 { t=a; a=c; c=t; } 
 if(b<c) 
 { t=b; b=c; c=t; } 
 printf("%6.2f,%6.2f,%6.2f\n",a,b,c); 
}
```

## 4 条件运算符

```
表达式1 ? 表达式2 : 表达式3
```

先计算 **表达式 1** 的取值：

如果表达式 1 的值为真（即非0），
则整个表达式取表达式 2 的值；

如果表达式1 的值为假。则整个表达式取表达式3 的值

- 条件运算符的优先级低于关系运算符和算术运算符，高于赋值符

- 条件运算符从左至右执行

例：` a>b? a:c>d?c:d `应理解为 ` a>b? a:(c>d?c:d) `

例： 字符转换，对输入的字符，判断是大写字母还是小写字母，如果是大写字母，就转换成小写字母；

```
#include <stdio.h>
void main ( ) 
{ char ch;
 scanf("%c",&ch);
 ch=(ch>='A'&& ch<='Z')?(ch+32):ch;
 printf("%c\n",ch);
}
```

```

{
```
